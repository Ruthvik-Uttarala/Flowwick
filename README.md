# Flowwick

Post once. Sell everywhere.

Flowwick helps boutique sellers create one product post and publish it to Shopify and Instagram.

## What it does

- Create and edit product posts in one place
- Enhance post copy with AI
- Publish to Shopify and Instagram
- Track post status from draft to posted
- Restore or permanently delete removed posts from Trash

## Tech stack

- Next.js (App Router)
- Supabase (auth, storage, data)
- Shopify Admin API
- Instagram Graph API

## Local setup

1. Copy `.env.example` to `.env.local`
2. Fill in Supabase, Shopify, Instagram, and OpenAI variables
3. Install dependencies and run:

```bash
npm install
npm run dev
```
